Okay, FINALLY, there is the option of a non-white Drunkit background. It's called the 'high contrast' theme and is white text on a darkish-blue background.

I've been using it for a while now, it's not perfect (some things don't quite look right) and there is less emphasis on visuals (i.e. it's not that pretty), but it's fairly easy to read, the text is slightly larger and there is contrast between text and most elements (i.e. you can see the difference between input fields and buttons easily).

Hope you like it. You can change to this theme by going to the theme control panel and selecting 'Drunkit Clean (High Contrast)'.
